# Weather App API

This is the backend module, see 
[weather_client](https://github.com/revillo/weather_client)
For client.

### Requirements
NodeJS + npm

PM2 (npm install -g pm2@latest)

### Install
```
npm install
```

### Deploy in production
```
npm run start:pm2
```
Runs on port 2052 by default

### Test
```
npm test
```
